# Email Performance Analysis: Top Countries

##  Goal
Analyze email campaign performance and account activity by country.

##  Description
This project analyzes email and account data to:
- aggregate account and email metrics by date and country,
- rank countries by number of accounts and sent emails,
- identify top 10 countries by email activity,
- analyze user behavior by verification and subscription status.

##  Tools
- SQL (BigQuery)
- Looker Studio (for visualization)
